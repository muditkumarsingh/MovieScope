# 🎬 MovieInfo — React + TypeScript Project

A fully responsive **Movie Information Website** built with **React**, **TypeScript**, and **Tailwind CSS**.  
This project fetches live data from the **TMDB API** to display movie details dynamically — with no local storage involved.

## 🛠️ Tech Stack
- **React.js** — for building dynamic and reusable UI components  
- **TypeScript** — for type safety and better code reliability  
- **Tailwind CSS** — for responsive and modern UI design  
- **TMDB API** — for fetching movie data (titles, images, ratings, etc.)  

---

## ⚙️ Features
✅ Fetches real-time movie data directly from TMDB API  
✅ Fully responsive layout using Tailwind CSS  
✅ Dynamic movie display with details and images  
✅ No local storage or hardcoded data — everything from the API  
✅ Clean code with TypeScript interfaces and modular structure
